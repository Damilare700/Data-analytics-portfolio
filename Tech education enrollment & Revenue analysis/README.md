# Tech Education Enrollment & Revenue Analysis

### **Project Overview**
Built an Excel dashboard analyzing 1,000 student enrollments across 7 tech courses to track revenue, course demand, student demographics, and academic performance. Designed for education program managers to monitor KPIs and optimize course offerings.

### **Scope**
Analyzed 1,000 student records from 2023–2025 covering courses in Cloud Computing, Cyber Security, Data Science, Machine Learning, Python, UI/UX Design, and Web Development. Tracked total revenue, enrollment sources, gender distribution, grades, and average fees.

### **Key Features**
- **KPI Dashboard**: Total Revenue ₦126.5M, Total Students 1,000, Avg Age 32, Top Course by Revenue
- **Pivot Analysis**: Revenue by Course, Enrollments by Gender, Grade Distribution, Monthly Enrollment Trends
- **Visuals**: Course enrollment bars, revenue by course, grade distribution, monthly enrollment line
- **Interactivity**: Slicers for Course, Gender, Grade, Enrollment Source, State
- **Demographic Analysis**: Gender split and age averages per course

### **Key Insights**
1. **Total Revenue**: ₦126,548,244 generated across all 7 courses
2. **Top Course by Revenue**: Web Development ₦20.11M, followed by Machine Learning ₦20.69M
3. **Most Popular Course**: Machine Learning 168 students, Web Development 158 students
4. **Gender Distribution**: Other 349, Male 329, Female 322 — nearly balanced
5. **Academic Performance**: 217 Grade A students, but 205 Grade F students need intervention
6. **Peak Enrollment**: September 96 students, October 93 students — Q3/Q4 strongest
7. **Average Age**: 32 years old, indicating working professionals upskilling

### **Tools & Techniques**
Excel, Power Query, Pivot Tables, Charts, Conditional Formatting, Data Modeling, KPI Design
